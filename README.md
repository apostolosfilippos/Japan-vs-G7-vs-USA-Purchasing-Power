



Data Sources and Methodology
All data used in this project was retrieved from the OECD Data Explorer. The primary datasets include:
   Average Annual Wages (Identifier: DSD_EARNINGS@AV_AN_WAGE) which provides data on average annual earnings per full-time equivalent employee.
   Annual GDP and Expenditure (Identifier: DSD_NAMAIN10@DF_TABLE1) specifically for Actual Individual Consumption (AIC) to measure household welfare.
   Annual National Disposable Income (Identifier: DSD_NAMAIN10@DF_TABLE2) to track the net income available to residents.
   Annual Purchasing Power Parities and Exchange Rates (Identifier: DSD_PPP@DF_PPP_INDICATORS) used to derive the Price Level Index (PLI).

To ensure a scientifically sound comparison between Japan and other G7 nations, all financial values have been normalized using Purchasing Power Parities (PPP) instead of nominal exchange rates. Furthermore, all series use Constant Prices to account for inflation, allowing for a clear view of real economic stagnation over the 2000-2024 period. The Price Level Index was manually calculated by dividing the PPP for actual individual consumption by the annual average exchange rate, multiplied by 100, following standard OECD statistical guidelines.
